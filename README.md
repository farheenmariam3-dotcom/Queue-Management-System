# Queue Management System

## Project Overview

The Queue Management System (QMS) is a web-based application designed to manage customer queues efficiently. It helps reduce waiting time, improve service quality, and provide a better customer experience through a digital token-based queue management process.

---

## Features

### 🎫 Token Generation
- Generates unique token numbers automatically
- Maintains organized customer flow

### 📺 Real-Time Queue Display
- Displays currently serving token
- Updates instantly when next token is called

### 👨‍💼 Admin Dashboard
- Manage queue operations
- Call next customer
- Monitor queue statistics

### 🔍 Token Search
- Search token number
- View current queue position

### 📊 Queue Statistics
- Total Tokens
- Waiting Tokens
- Completed Tokens

### 🌙 Dark Mode
- Toggle between light and dark theme

### 🔔 Notifications
- Displays alerts when a token is called

---

## System Workflow

1. Customer generates a token
2. Token enters waiting queue
3. Customer waits for their turn
4. Administrator calls next token
5. Current token is displayed
6. Customer receives service
7. Queue moves to next customer
